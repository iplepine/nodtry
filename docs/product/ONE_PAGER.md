# Nodtry 한 장 요약

마지막 갱신: 2026-05-30 · 최신 출시 빌드 `1.0.47+69`

## 한 줄 정의
Nodtry(그래,해봐)는 혼자 못 지키는 작은 약속을 가까운 사람과 묶어, 미루는 사용자가 도망가지 못하게 만드는 1:1 상호 책임 앱이다.

## 풀려는 문제
- 개인 목표 앱은 기록은 쉽지만 포기는 더 쉽다. 알림은 무시되고, 실패는 혼자 묻힌다.
- 의지가 약한 사용자는 더 예쁜 루틴 관리가 아니라 누군가에게 걸려 있는 약속 구조가 필요하다.
- 가까운 사람의 압박은 효과적이지만 타이밍과 역할이 애매해서 일상에서 지속되기 어렵다.

## 핵심 사용자
- 혼자 루틴을 유지하기 어렵고 스스로를 못 믿는 성인.
- 가까운 사람(연인·친구·가족)과 서로 응원하며 습관을 만들고 싶은 사람.
- 학습·운동·회복·생활관리처럼 반복 실천이 중요한 사람.
- 1차 vertical: 성인 학습자(4주 스터디 책임 스프린트).

## 목표
- **단기**: 1:1 파트너 기반 실천 루프(연결 → 계획 → 실행 → 확인 → 정산)가 끝까지 작동하는지 검증.
- **MVP 성공 기준**:
  - 신규 사용자가 10분 안에 파트너 연결 + 첫 플랜 생성을 완료한다.
  - 첫 주에 연결 사용자 40% 이상이 3회 이상 실천 기록을 남긴다.
  - 파트너 확인/리액션을 받은 사용자의 다음 실천율이 미수신 사용자보다 높다.
- **장기 방향**: 1:1에서 다대다 책임망으로 확장(`docs/product/RESPONSIBILITY_NETWORK.md`).

## 작동 루프
1. **연결** — 초대 코드로 1:1 파트너를 연결한다. 양방향 관계 1쌍만 허용.
2. **계획** — 카테고리(공부/운동/직접) → 추천 약속 → 요일·알림·설명 4단계로 작은 실천 플랜을 만든다. 파트너 미연결 시 즉시 active, 연결 시 파트너 승인 후 active.
3. **실행** — 지금 탭 상단 카드(오늘 할 일/지연/완료/빈 상태) 좌우 스와이프 캐러셀에서 "했어/휴식/건너뜀" 처리. 즉석 집중 타이머로 "지금 할게!"도 가능.
4. **확인** — 파트너는 상대 실천을 확인하고 리액션·코멘트를 보낸다. 약속 시간이 지나면 미실천이 묻히지 않고 파트너에게 자동 전달, 파트너는 "똑똑"으로 다시 당길 수 있다.
5. **정산** — 보상/벌칙 약속을 미리 합의해두면 기간 종료 시 완료 일수 기준으로 자동 정산하고 결과 카드로 양측에 노출한다.
