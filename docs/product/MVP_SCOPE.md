# MVP 범위

## 목표
1:1 파트너 기반 실천 루프가 끝까지 작동하는지 검증한다.

## 포함
- 이메일/기본 인증과 프로필
- 초대 코드 기반 1:1 파트너 연결
- 양방향 상호 지지 관계
- 4단계 플랜 생성: 스터디 템플릿, 행동, 요일, 알림, 설명
- 플랜 승인, 조율, 수정, 중단, 완료
- 지금 탭 카드: 오늘 할 일, 완료, 지연, 빈 상태, 파트너 상태
- 히스토리: 완료, 건너뜀, 확인, 리액션, 메모
- 똑똑 알림과 파트너 리마인드
- 보상/벌칙 약속 제안과 정산
- 로컬 알림과 주요 원격 알림
- 4주 스터디 파일럿 기본값: 주 3회/5회, 저녁 9시, 28일 기간

## 제외
- 다인 그룹
- 공개 피드
- 결제와 구독
- 코칭 마켓플레이스
- AI 코치
- 복잡한 통계 대시보드

## MVP 성공 기준
- 신규 사용자가 10분 안에 파트너 연결과 첫 플랜 생성을 완료한다.
- 첫 주에 연결 사용자 40% 이상이 3회 이상 실천 기록을 남긴다.
- 파트너 확인 또는 리액션을 받은 사용자의 다음 실천율이 미수신 사용자보다 높다.
