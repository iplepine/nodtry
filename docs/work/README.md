## 운영 규칙

- active task는 최대 3개만 둔다.
- task는 반드시 roadmap에 연결한다.
- roadmap은 반드시 goal에 연결한다.
- 완료 기준이 없는 task는 active에 두지 않는다.
- 개발 task는 테스트, 빌드/검증, 문서 업데이트 체크를 포함한다.
