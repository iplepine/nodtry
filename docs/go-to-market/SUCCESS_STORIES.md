# 성공 사례

## 사례 작성 기준
- 4주 전후 변화가 분명해야 한다.
- 실천 행동, 파트너 역할, 유지 계기가 포함되어야 한다.
- 과장된 성과보다 작지만 반복 가능한 변화를 기록한다.

## 템플릿
### 제목
사용자 유형과 변화 한 줄

### 시작 상황
- 어떤 행동을 유지하기 어려웠는가
- 왜 혼자서는 실패했는가

### Nodtry 사용 방식
- 플랜 내용
- 파트너와의 관계
- 알림, 확인, 똑똑, 약속 사용 방식

### 결과
- 4주 동안의 완료 일수
- 파트너 피드백 횟수
- 다음 플랜 생성 여부

### 인용
사용자가 직접 말한 짧은 문장 1~2개를 남긴다.

## 초기 목표 사례
- 친구와 주 3회 걷기
- 커플의 수면 준비 루틴
- 스터디 파트너의 영어 공부
- 가족 간 물 마시기 또는 스트레칭
