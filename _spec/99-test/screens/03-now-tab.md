# 화면별 TC - 홈: Now 탭 (Now Tab)

**화면**: `HomeScreen` > `NowTab`

| ID | 케이스명 | 사전 조건 | 액션 | 기대 결과 | 비고 |
|:---:|:---|:---|:---|:---|:---|
| N-01 | **카드 없음 (계획 필요)** | 활성 계획 없음 | 화면 진입 | Plan Needed(Type 2-1) 카드 노출 & '계획 만들기' 버튼 표시 | |
| N-02 | **지금 실천 (Now Action)** | 현재 시간대 실천 계획 있음 | '했어' 버튼 탭 | 완료 애니메이션 동작, DB 업데이트, 카드가 사라지거나 상태 변경 | |
| N-03 | **오늘 완료 (Today Done)** | 오늘 모든 할 일 완료 | 화면 확인 | "오늘 다 했어요"(Type 2-2) 카드 노출 | |
| N-04 | **파트너 소식 확인** | 파트너가 실천함 | 화면 상단/중앙 확인 | Partner Action Share(Type 4) 카드 노출, 프로필 이미지 확인 | |
| N-05 | **파트너 소식 반응** | Partner Action Share 카드 있음 | '확인하기' 버튼 탭 | "함께하는 중" 등으로 상태 변경, 버튼 비활성화 또는 카드 변경 | |
| N-06 | **디버그 시나리오 전환** | Mock 모드 빌드 | 우측 하단 🐞 FAB 탭 | 시나리오 선택 다이얼로그 노출 및 선택 시 UI 즉시 반영 | |
