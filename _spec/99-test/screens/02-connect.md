# 화면별 TC - 연결 화면 (Connect)

**화면**: `ConnectScreen` (로그인 후 파트너 미연결 시 접근, 또는 설정에서 접근)

| ID | 케이스명 | 사전 조건 | 액션 | 기대 결과 | 비고 |
|:---:|:---|:---|:---|:---|:---|
| C-01 | **초대 코드 생성** | 미연결 유저 | '초대 코드 생성' 버튼 탭 | 6자리 또는 8자리 초대 코드 표시 & 공유하기 시트 노출 | |
| C-02 | **초대 코드 입력 (유효)** | 미연결 유저 | 상대방의 유효한 코드 입력 | 연결 성공 팝업 -> `HomeScreen` 이동 | 상대방 프로필 정보 표시 |
| C-03 | **초대 코드 입력 (오류)** | 미연결 유저 | 잘못된/만료된 코드 입력 | 에러 토스트 메시지 노출 | |
