# 화면별 TC - 계획 생성 (Plan Create)

**화면**: `PlanCreateScreen`

| ID | 케이스명 | 사전 조건 | 액션 | 기대 결과 | 비고 |
|:---:|:---|:---|:---|:---|:---|
| P-01 | **단계 이동** | - | '다음' 버튼 탭 | 다음 단계(제목 -> 시간 -> 요일 등)로 슬라이드 이동 | |
| P-02 | **필수 입력 체크** | - | 입력 없이 '다음' 탭 | 버튼 비활성화 상태 또는 유효성 검사 에러 | |
| P-03 | **계획 생성 완료** | 마지막 단계 | '완료' 버튼 탭 | 로딩 후 `NowTab`으로 이동, 새 계획 카드 표시 | |
