# Now 탭 테스트 시나리오 가이드

## 1. 테스트 목적
Now 탭의 5가지 핵심 카드 타입과 복합적인 상황들이 UI에 올바르게 반영되는지 검증합니다.

## 2. 테스트 환경 설정
- **기본**: 이 기능은 `MockRecordRepository`가 활성화된 상태에서만 동작합니다. (개발 환경 기본값)
- **진입 방법**: Now 탭 화면 우측 하단의 **버그 아이콘(🐞) FAB**를 탭하여 시나리오를 선택할 수 있습니다.

## 3. 테스트 시나리오 목록 (MockScenario)

### 기본 상태
1. **`planNeeded` (Type 2-1)**
   - **상황**: 등록된 계획이 전혀 없는 경우
   - **기대 결과**: "지금은 약속이 없어요" 카드 표시 + 계획 생성 유도 버튼

2. **`relaxedDay` (Type 2-2 B)**
   - **상황**: 계획은 생성되어 있으나, 오늘(현재) 실천할 항목이 없는 경우
   - **기대 결과**: "오늘은 여유로운 날이에요" 메시지 + 하단(Secondary)에 위치

3. **`todayDone` (Type 2-2 A)**
   - **상황**: 오늘 할 일을 모두 완료한 경우
   - **기대 결과**: "오늘 다 했어요" 긍정 메시지 + 체크 아이콘이 포함된 카드

### Executor (내 실천)
4. **`nowActionMorning` / `Afternoon` / `Night` (Type 1)**
   - **상황**: 현재 시간에 맞춰 실천해야 할 계획이 있음
   - **기대 결과**: 화면 중앙에 크고 명확한 Action Card 표시, "했어" 버튼 활성화

5. **`overdueSelfAction` (Type 5)**
   - **상황**: 실천 시간이 지났지만 아직 완료 처리를 안 함
   - **기대 결과**: 부드러운 톤의 "지난 실천" 카드 표시, Secondary 영역 또는 Primary 하단 노출

### Manager & Interaction (파트너)
6. **`partnerPlanShare` (Type 3)**
   - **상황**: 파트너가 새로운 계획을 생성함
   - **기대 결과**: "이런 약속을 제안했어요" 헤더 + 파트너 프로필 이미지 표시

7. **`partnerActionShare` (Type 4)**
   - **상황**: 파트너가 실천을 완료함
   - **기대 결과**: "오늘도 완료했어요!" 헤더 + 파트너 프로필 + "확인하기" 버튼

### 복합 시나리오 (Complex)
8. **`partnerActionShareWithNowAction`**
   - **상황**: 내가 할 일(Now Action)도 있고, 파트너의 완료 소식(Partner Action)도 도착함
   - **기대 결과**: 
     - 상단: 내 실천 카드 (Primary)
     - 하단: 파트너 소식 카드 (Manager Quick Card) - 파트너 프로필 포함

9. **`partnerPlanShareWithNowAction`**
   - **상황**: 내가 할 일도 있고, 파트너의 새 제안도 도착함
   - **기대 결과**: 위와 유사하게 두 카드가 모두 적절히 배치되어야 함

## 4. 검증 포인트
- **UI**: 겹침 현상 없이 스크롤 가능한가?
- **Logic**: 각 버튼(`했어`, `확인하기`) 클릭 시 스낵바 및 상태 변화가 일어나는가?
- **Context**: 파트너 카드의 경우 프로필 이미지가 올바르게 노출되는가?
