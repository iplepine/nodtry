# 화면 기능 명세: 스플래시 및 로그인 (Splash & Login)

## 1. 개요
앱의 진입점으로, 사용자 인증 상태를 확인하고 적절한 홈 화면으로 안내하거나 회원가입 프로세스를 진행한다.

> 상세 로직: [01-splash-login.md](../01-splash-login.md)

---

## 2. 제공 기능 목록 (Feature List)

### 2.1 진입 및 인증 분기
1. **자동 로그인 (Auto Login)**
   - 기존 세션 및 토큰 유효성을 확인하여 즉시 홈으로 이동한다.
2. **버전/설정 로드 (Bootstrap)**
   - 앱 구동에 필요한 기본 설정을 불러온다.

### 2.2 회원가입/로그인 유형
3. **게스트 로그인 (Guest Login)**
   - 별도 가입 절차 없이 로컬 계정으로 즉시 체험한다.
4. **구글 로그인 (Google Login)**
   - 구글 계정을 연동하여 가입 또는 로그인한다.
5. **이메일 로그인 (Email Login)**
   - 이메일/비밀번호 방식을 통해 가입 또는 로그인한다. (현재 Legacy/Sub 옵션)
