# 화면 기능 명세: Us 탭 (Us Tab)

## 1. 개요
나와 파트너가 현재 진행 중인 모든 계획(Plan)을 한눈에 모아보고 관리하는 공간이다.
진행 중인 계획의 목록을 확인하고, 새로운 계획을 추가할 수 있다.

> 상세 로직: [03-03-us-tab.md](../03-03-us-tab.md)

---

## 2. 제공 기능 목록 (Feature List)

### 2.1 계획 목록 조회
1. **내 계획 보기 (My Plans)**
   - 내가 실행 중인 계획 리스트를 확인한다.
2. **파트너 계획 보기 (Partner Plans)**
   - 파트너가 실행 중인 계획 리스트를 확인한다.
3. **진행 상태 요약 (Progress Summary)**
   - 각 계획의 주간 달성률이나 현재 상태(Active/Paused)를 개괄적으로 본다.

### 2.2 계획 관리
4. **새 계획 생성 진입 (Create New Plan)**
   - 새로운 목표를 설정하기 위한 작성 화면으로 이동한다.
5. **계획 상세/수정 (Plan Details)**
   - (TODO) 특정 계획을 탭하여 상세 정보를 보거나 수정을 요청한다.
