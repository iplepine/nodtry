# 화면 기능 명세: 기록 탭 (Record Tab)

## 1. 개요
과거의 실천 이력(History)을 타임라인 형태로 조회하고, 누락된 기록을 관리하는 공간이다.

> 상세 로직: [03-02-history-tab.md](../03-02-history-tab.md)

---

## 2. 제공 기능 목록 (Feature List)

### 2.1 이력 조회
1. **타임라인 뷰 (Timeline View)**
   - 날짜별로 정렬된 나와 파트너의 실천 기록을 스크롤하여 확인한다.
2. **검증 여부 확인 (Verification Status)**
   - 각 기록이 파트너에 의해 확인(Verified)되었는지 여부를 아이콘 등으로 식별한다.

### 2.2 기록 보정
3. **미완료 기록 소명 (Reconcile Past)**
   - 시스템에 의해 'Skipped' 처리된 항목을 선택하여 사유를 남기거나 뒤늦게 완료 처리한다.
