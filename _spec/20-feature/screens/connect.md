# 화면 기능 명세: 연결 (Connect)

## 1. 개요
나와 함께 목표를 공유하고 책임을 나눠가질 '파트너'와 연결하는 과정이다.

> 상세 로직: [02-connect.md](../02-connect.md)

---

## 2. 제공 기능 목록 (Feature List)

### 2.1 초대
1. **초대 코드 생성 (Generate Invite Code)**
   - 내 고유 연결 코드를 생성하고 화면에 표시한다.
2. **코드 공유 (Share Code)**
   - 생성된 코드를 클립보드 복사하거나 메신저로 공유한다.

### 2.2 연결
3. **초대 코드 입력 (Enter Code)**
   - 상대방으로부터 전달받은 코드를 입력하여 검증한다.
4. **연결 확정 (Confirm Connection)**
   - 유효한 코드 입력을 통해 파트너 관계를 맺고 홈 화면으로 이동한다.
