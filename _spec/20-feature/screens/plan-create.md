# 화면 기능 명세: 계획 생성 (Plan Create)

## 1. 개요
새로운 행동 목표를 설정하고, 실행 주기와 알림 등을 구체화하여 시스템에 등록하는 마법사(Wizard) 화면이다.

> 상세 로직: [05-01-plan-creation-ui.md](../05-01-plan-creation-ui.md)

---

## 2. 제공 기능 목록 (Feature List)

### 2.1 단계별 입력
1. **목표 설정 (Title Input)**
   - 무엇을 할 것인지 텍스트로 정의한다.
2. **시간 설정 (Time Selection)**
   - 언제 수행할 것인지(구체적 시간 or 모호한 시간대)를 선택한다.
3. **요일/빈도 설정 (Day/Frequency)**
   - 일주일에 몇 번, 무슨 요일에 할 것인지를 정한다.

### 2.2 검토 및 완료
4. **미리보기 (Summary Preview)**
   - 입력한 내용을 종합하여 최종 확인한다.
5. **생성 완료 (Submit)**
   - 계획을 DB에 저장하고 활성화한다.
