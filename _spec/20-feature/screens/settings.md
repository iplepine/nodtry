# 화면 기능 명세: 설정 (Settings)

## 1. 개요
앱의 전반적인 환경 설정, 계정 관리, 정보 조회를 담당한다.

---

## 2. 제공 기능 목록 (Feature List)

### 2.1 계정 관리
1. **로그아웃 (Logout)**
   - 현재 세션을 종료하고 로그인 화면으로 돌아간다.
2. **회원 탈퇴 (Withdraw)**
   - 계정 및 모든 데이터를 영구 삭제한다.
3. **계정 연결 (Account Link)**
   - (게스트인 경우) 정식 계정(Google 등)으로 전환/연결한다.

### 2.2 환경 설정
4. **언어 변경 (Change Language)**
   - 앱의 UI 언어를 한국어/영어 등으로 변경한다.
5. **테마 변경 (Change Theme)**
   - (옵션) 다크모드 또는 앱의 컬러 테마를 변경한다.

### 2.3 정보
6. **버전 정보 (Version Info)**
   - 현재 앱의 버전을 확인한다.
